About

PotatoDB is a Node.js library for building and managing JSON databases in the local file system of node.js servers. PotatoDB provides functions and methods for creating and managing databses, inserting, querying, updating, and deleting data from the databases. PotatoDB also allows you to manage multiple databases in the same server.


PotatoDB has a very basic and simple usage, and is very easy to deal with. It also introduces simplicity and flexibility in querying and updating data, which could be challenging to implement with other database management systems.
